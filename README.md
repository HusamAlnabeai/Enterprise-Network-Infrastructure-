# Enterprise Network Infrastructure | CCNA Lab Project

A comprehensive Cisco Packet Tracer project that simulates a real-world enterprise network infrastructure. This lab was developed to practice and demonstrate core CCNA networking concepts through hands-on configuration, testing, and troubleshooting.

---

## Project Overview

This project represents a multi-site enterprise network built using Cisco Packet Tracer. It includes routing, switching, network segmentation, security, and essential network services to simulate a production-like environment.

The objective of this project is to strengthen practical networking skills by implementing industry-standard technologies and validating network functionality through testing and troubleshooting.
